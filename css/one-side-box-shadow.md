# one side box shadow

We just want to one side which has box shadow sometime.

### example 1

```css
    .box{
        width:100px;
        height:100px;
        background-color:#fff;
        box-shaow:inset 0 -10px 10px -10px #000000;
    }

```


### example 2

```css
    
    .box2{
        width:100px;
        height:100px;
        margin:30px;
        background-color:#fff;
        box-shadow:0 0 3px rgb(0,0,0), 0 7px 3px rgba(0,0,0,.8);   
    }

```
